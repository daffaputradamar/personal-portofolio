**Ecosystem** is a web platform designed to empower sales teams by streamlining how they share product information. Salespeople can easily browse and share products managed by the admin, enabling faster and more professional client engagement.

## Features

- **Product Sharing**: Sales reps can instantly share product details with clients via personalized links.
- **Admin Panel**: Admins manage product data including descriptions, images, and categories.
- **Mobile-First Design**: Built with responsiveness in mind, perfect for use on the go.
- **Search & Filter**: Quickly find the right product with smart search and category filters.

## Technologies

- **NextJs**: Delivers a fast, SEO-friendly frontend with server-side rendering.
- **Tailwind CSS**: Ensures consistent, modern styling across all pages.
- **Vercel**: For hassle-free deployment and performance optimization.
- **Shadcn/UI**: Accessible and reusable component library for sleek UI.

## Getting Started

To run the app locally:

```bash
git clone https://github.com/daffaputradamar/ecosystem-bmw
cd ecosystem-bmw
npm install
npm run dev
```

## Deployment

The project is deployed using Vercel for automatic builds, preview environments, and fast global delivery.
