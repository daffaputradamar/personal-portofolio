**BidikASN** is an online ASN (CPNS, PPPK, Sekdin) tryout platform built to help aspiring civil servants practice and evaluate their readiness. With a modern and responsive interface, BidikASN delivers a smooth user experience across devices.

## Features

- **Tryout System**: Simulate real CPNS exams with time limits and randomized questions.
- **Result & Ranking**: Automatically graded tests with instant scores and user ranking.
- **Question Management**: Admin panel to manage question banks, categories, and exams.
- **Responsive UI**: Works great on both desktop and mobile.

## Technologies

- **NextJs**: For fast, scalable frontend and server-side rendering.
- **Tailwind CSS**: For modern, responsive styling.
- **Vercel**: For seamless deployment with CI/CD support.
- **Shadcn/UI**: Beautiful UI components for better accessibility and design consistency.
