**Psikotes Kuesioner** is a web-based questionnaire platform designed for collecting and analyzing survey responses. With a simple and responsive design, it enables participants to go through a series of questions smoothly, while providing admins with tools to manage and review results.

## Features

- **Questionnaire Flow**: A clean, step-by-step question interface with multiple-choice answers.
- **Admin Dashboard**: View, export, and manage all survey responses, complete with user metadata.
- **Progress Indicator**: Visual feedback on how far along the user is in the survey.
- **Real-time Export**: Export individual or all user responses directly to Excel format.

## Technologies

- **HTML + Vanilla JavaScript**: Lightweight and fast frontend built without any frontend frameworks.
- **Tailwind CSS**: Utility-first CSS framework for fast and consistent UI styling.
- **Firebase Hosting**: Fast and secure static hosting provided by Google Firebase.
- **Firebase Realtime Database / Firestore**: (Optional) backend storage for user responses and auth.

## Getting Started

To set up the project locally:

```bash
git clone https://github.com/daffaputradamar/kuisioner-web
cd psikotes-kuesioner
# Open index.html with a live server extension or serve it via HTTP
```

## Deployment

This project is deployed on Firebase Hosting. To deploy your own version:

```bash
npm install -g firebase-tools
firebase login
firebase init
firebase deploy
```

## Conclusion

Psikotes Kuesioner is ideal for teams, psychologists, or researchers who want a simple and efficient way to distribute and collect questionnaire data online. Whether you're conducting a survey or running a psychological test, this app delivers a clean and straightforward experience.
