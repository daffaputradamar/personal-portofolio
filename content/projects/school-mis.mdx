**School Management Information System** is a comprehensive web application developed using **PHP CodeIgniter**. It helps school administrators manage key academic operations digitally, including data on students, classes, teachers, subjects, and public news updates.

## Features

- **Dashboard**: At-a-glance overview of school metrics like number of students, teachers, and classes.
- **Class & Subject Management**: Create, edit, and assign subjects to classes with ease.
- **Teacher Directory**: Manage teacher profiles and their class assignments.
- **News & Announcements**: Easily publish and manage school news and events.
- **Responsive UI**: Clean and accessible layout designed for both desktop and mobile users.

## Technologies

- **PHP (CodeIgniter)**: Lightweight and powerful PHP framework for MVC development.
- **MySQL**: Relational database for structured academic data.
- **Bootstrap**: For a consistent and responsive frontend layout.
- **jQuery**: Enhances interactivity and dynamic behavior.

## Getting Started

Clone the project and set it up on your local server:

```bash
git clone https://github.com/daffaputradamar/si-sekolah
```

Configure your .env or application/config/database.php for database access.

Import the provided SQL file into your MySQL server, then:

```bash
php spark serve
# or access through localhost if using XAMPP/Laragon
```


## Deployment

This app can be hosted on any PHP server, like shared hosting, VPS, or a local server stack like XAMPP or Laragon.

## Conclusion

This school information system simplifies academic management and improves communication between school staff and students. It's a valuable tool for schools looking to digitize operations and modernize their infrastructure.
